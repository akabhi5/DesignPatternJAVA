- Creational
- Structural
- Behavioural

| Singleton        | Adapter    | Template Method         |
| Factory          | Composite  | Mediator                |
| Abstract Factory | Proxy      | Chain of Responsibility |
| Builder          | Fly weight | Observer                |
| Prototype        | Facade     | Strategy                |
| Factory          | Bridge     | Command                 |
|                  | Decorator  | State                   |
|                  |            | Visitor                 |
|                  |            | Iterator                |
|                  |            | Interpreter             |
|                  |            | Memento                 |
